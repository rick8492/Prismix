Elementos del prisma

Interesantes elementos dentro del prisma.

Ver

Caras

Son como las paredes de una caja. Imagina un cubo o una caja de zapatos: tiene una parte de arriba, una parte de abajo y cuatro paredes laterales. En un prisma, estas paredes son las caras.

Aristas

Son como las líneas que conectan las esquinas de las paredes. Si imaginas las esquinas de una caja, las líneas que conectan esas esquinas son las aristas.

Vertices

Son simplemente las esquinas donde se encuentran las aristas. Cada esquina de la caja es un vértice.

Altura

Es lo alto que es el prisma, desde la parte de arriba hasta la parte de abajo. Puedes imaginarlo como la altura de una caja de zapatos.

Area de la base

Imagina que estás decorando una caja para regalo. Si esa caja tiene forma de prisma rectangular, para calcular la cantidad de papel de regalo que necesitas, simplemente mides el área de las dos bases y la suma de las áreas de las caras laterales. ¡Así podrás envolverla perfectamente!

Area Lateral

Es la superficie de las paredes laterales del prisma. Imagina desdoblar la caja y extenderla. La superficie que verías es el área lateral.
